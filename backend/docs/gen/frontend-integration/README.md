## 关键决策点

在开始实施前，需要确认以下技术选择:

1. **认证方式**: JWT Token / Cookie Session / 混合方案
2. **字段映射策略**: 后端适配前端 / 前端适配后端 / 协商统一
3. **路由路径调整**: 添加别名 / 统一修改 / 前端适配
4. **历史对话管理**: 单 Thread / 多 Thread 聚合 / 单 Thread + 清空

详见 `analysis-and-planning.md` 第三章。
